# Bailiwick User Stories

As a Poster, I want to Publish a Post, so that my Friends can see it.


As a User  I want to log in, so that I can use the app.


As a User who has forgotten my password I want to be able to recover my private key so that I can use my account.


As a User I want to see new Posts from my Friends So that I know what they're up to.


As a Poster I want to control who can see my Post so that I retain (some) control of my data.


As a Poster I want to delete a Post so that no one can read it


As a Poster I want to edit a Post so that I can fix typos, etc


As a Poster I want to make a Post Public. so that anyone can read it.


As a Poster I want to be protected from brigading so that coordinated groups of strangers cannot attack me.


As a User I want to be able to access my account on multiple devices so that I can check my account from anywhere


As a Poster I want to Post text so that I can communicate in my preferred fashion


As a Poster I want to Post images so that I can communicate in my preferred fashion


As a Poster I want to Post video so that I can communicate in my preferred fashion


As a User I want to be able to find my IRL friends so that I can connect with them


As a User I want to be able to hide my profile data so that I can remain anonymous


As a User I want to be able to subscribe to other Users so that I can see what they Post.


As a User I want to be able to comment on a Post so that I can express my thoughts


As a User I want to be able to enter a Friendship with another User, wherein we are mutually subscribed to each other's Posts


As a Poster I want to organize my Subscribers/Friends into different groups so that I can publish a Post to specific subsets


As a Poster I want to be able to put a Subscriber/Friend into multiple groups so that I can publish to "all contacts" or "car nuts" or "friends" and know that the same person is in all the groups I need them in.


As a Poster I want to be able to send a file to any given person so that I can share files seamlessly.


As a User I want to limit the amount of data stored on my device so that I don't run out of space


As a User I want to control the types of content I download so that I don't see types of content I don't wish to.


As a User I want to be able to report NSFW/L content so that my friend circle isn't surprised


As a User I want to subscribe to my friends' block lists so that their recommendations can filter my content


As a User I want to override a blocklist so that I can view content that my squeamish friends have blocked


As a Poster I want to be able to tag the type of content I am Posting so that downstream Users can automatically filter things if they wish


As a User, I want to be able to see Subscription requests from other Users so that I can decide whether to allow another User to see my Posts.
